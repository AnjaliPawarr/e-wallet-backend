# 💳 E-Wallet Backend Service

A Spring Boot based **E-Wallet Backend Application** designed using clean architecture principles.  
This project simulates core wallet functionalities similar to Paytm/PhonePe at a backend level.

---

## 🚀 Features

- Create wallet for a user
- Add money to wallet
- Fetch wallet balance
- RESTful API design
- Database persistence using JPA
- Dockerized backend service
- GitHub version control

---

## 🛠️ Tech Stack

- **Java 17**
- **Spring Boot**
- **Spring Data JPA**
- **REST APIs**
- **MySQL / H2 (configurable)**
- **Docker**
- **Maven**
- **Git & GitHub**
